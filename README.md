# Karnaugh Map Studio

An interactive static website for electronics engineering students to learn Karnaugh map simplification from 2 to 6 input variables.

## What is included

- A practice-focused landing page for Karnaugh maps
- An interactive 2-, 3-, 4-, 5-, and 6-variable K-map editor with `0`, `1`, and `X` states
- Automatic simplification into minimal SOP and POS forms
- Guided explanations for why each group produces its term
- Preset examples for pairs, quads, octets, wrap-around groups, and don't-care usage
